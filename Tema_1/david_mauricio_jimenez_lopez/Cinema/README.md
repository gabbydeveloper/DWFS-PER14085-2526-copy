## Diseño y Herramientas

El diseño inicial de esta aplicación fue conceptualizado utilizando **Stitch de Google** (https://stitch.withgoogle.com/), una herramienta de IA para la generación de interfaces web.

### Prompt Utilizado para el Diseño

```
Diseña una página web moderna para un sistema de reserva de butacas de cine llamado "UNIR-CINEMA". 

Requisitos específicos:
- Fondo oscuro 
- Representación visual de una pantalla de cine en la parte superior
- Grid de 5 filas × 12 asientos con estados visuales claros:
  * Disponibles: color verde con hover interactivo
  * Ocupados: color rojo con símbolo X
  * Seleccionados: color dorado con animación de pulso
- Ayuda explicativa de los estados de asientos
- Formulario de reserva con:
  * Selector de cantidad de asientos
  * Display de asientos seleccionados
  * Calculadora de precio total
  * Botón de confirmación estilizado
- Footer con información de contacto del cine
- Paleta de colores: negro, gris oscuro, verde, rojo, dorado (#ffc107)
- Tipografía moderna y legible
- Efectos visuales sutiles: sombras, gradientes y animaciones CSS
- Estilo premium que refleje la experiencia cinematográfica

El diseño debe ser elegante, funcional y transmitir la atmósfera de un cine moderno.
```
